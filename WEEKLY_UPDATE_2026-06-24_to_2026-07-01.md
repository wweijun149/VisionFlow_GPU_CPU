# AOI CV Based 週更新紀錄

期間：2026-06-24 到 2026-07-01

這份是給流水帳報告用的更新紀錄，內容用比較簡單的方式整理。

## 2026-06-24

- 新增全域 logging 系統，讓 CLI、GUI、批次處理、監控模式和報表輸出都可以留下執行紀錄。
- GUI 新增資料夾監控模式，可以監看資料夾裡新增的圖片，並自動一張一張檢測。
- Batch Dashboard 新增 tile scatter 圖，可以用圖表看每張圖的 tile PASS / NG 分布。
- Monitor 模式也加入 scatter 圖，方便看即時處理結果。
- 完成 AOI_GUI v0.1.0 Windows 執行檔打包與 Release。
- 新增獨立 scatter plot 匯出工具，可以從 JSON 或 CSV 報表產生圖表。
- scatter 匯出工具加入 JSON / CSV 類型篩選，報表軸線與標籤也做了整理。
- 新增 matrix CSV 輸出，讓檢測結果可以用 row / column 矩陣方式查看。

## 2026-06-25

- 準備 AOI_GUI v0.2.0 Windows 版本，主要包含 scatter 匯出與 matrix CSV 更新。
- 新增 matrix CSV summary 匯出工具，可以選資料夾後彙整多份檢測結果。
- 調整 matrix summary 格式，讓 id 排序和 NG 標記更穩定。
- Monitor 模式新增右鍵功能，可以開啟已處理的原圖。
- Monitor 模式新增處理完成後搬移圖片的功能，並保留原本資料夾結構。
- 修正 popup menu 和 dialog 的顏色，讓文字比較容易看。
- 完成 AOI_GUI v0.3.0 Windows 執行檔打包與 Release。
- 新增 `PROJECT_REPORT.md`，整理專案功能、流程、使用情境、輸出項目和後續方向。

## 2026-06-26

- NG tile 輸出圖新增紅色 defect box，方便直接看出問題位置。
- 將包裝後的產品名稱調整為 VisionFlow AOI。
- 準備 VisionFlow AOI v1.0.0 Windows 版本與打包設定。
- 新增 Detector401-2 檢測項目，支援最小灰階規則。
- Detector401-2 再加入白色比例規則，用來判斷 NG 條件。
- 更新對應 recipe 和 GUI detector 名稱顯示。

## 2026-06-29

- Monitor 模式新增累積 tile scatter 圖，可以把多張圖片的檢測結果堆疊顯示。
- 調整 Monitor scatter 圖的排列方式，讓不同圖片的資料比較容易分開看。
- GUI 新增模式權限切換，包含 OP、Engineer、Admin。
- OP 模式只保留監控操作，減少現場使用時誤改設定的機會。
- Engineer 模式限制部分進階 detector 參數，只保留比較常用的幾何類設定。

## 2026-06-30

- 新增 Detector900，用來檢查雙框間距與外框、內框候選。
- 新增 Detector900 對應 recipe。
- Detector900 加入 frame spacing 檢查，確認內外框距離是否符合條件。
- 調整 Detector900 的 frame 面積篩選方式。
- 新增 Detector900 NG tile debug overlay，可以在 NG tile 上看到候選框、原因文字和邊距資訊。
- 加入 rejected candidate 顯示，方便確認為什麼某些候選框被排除。
- 最後將 Detector900 改成只用外框 / 內框的寬高篩選，不再依賴面積限制。

## 2026-07-01

- 整理 2026-06-24 到 2026-07-01 的更新內容，新增這份週更新紀錄。

## 本週重點整理

- 報表輸出變多：新增 scatter 圖、matrix CSV、matrix summary。
- Monitor 模式更完整：支援資料夾監控、右鍵開原圖、完成後搬移圖片、累積 scatter 圖。
- GUI 權限更清楚：新增 OP / Engineer / Admin 模式。
- 打包版本持續整理：完成 AOI_GUI v0.1.0、v0.2.0、v0.3.0，以及 VisionFlow AOI v1.0.0 準備。
- Detector 持續增加：新增 Detector401-2 和 Detector900，並補上對應 recipe 與 debug 顯示。
